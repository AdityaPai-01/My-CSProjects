MAIN FUNCTIONS: 
1. Add Tasks  --at
2. Remove Tasks  --rt
3. Edit Tasks: --et
    - Parameters: TaskName, TaskDescription
4. Show Tasks:
    - All Tasks --sa
    - Details of specific Tasks --sd
5. Create new List  --nl
6. Change current list --cl

About the files:
1. keeplogin.json --> It keeps the data of the user which has turned on keep login. Basically, no need to login again when you open the app after you close it.
2. regusers.json --> It keeps data about all the users registered in the application database. For now, just the username and password.
